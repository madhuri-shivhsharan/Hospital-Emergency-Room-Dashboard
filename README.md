# Hospital-Emergency-Room-Dashboard
This project is an interactive Excel dashboard for a hospital emergency room. It helps to analyze patient data, waiting time, and hospital performance in a simple and visual way.

---

## 📈 Key Details
- Total Patients: 488  
- Average Wait Time: 35.20 minutes  
- Satisfaction Score: 4.79  

---

## 📊 Features
- Monthly and Year filters  
- Patient status (On-time vs Delay)  
- Gender report  
- Admission status  
- Age group analysis  
- Department referral  

---

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts  

---

## 💡 Insights
- 55% patients admitted  
- 41% patients delayed  
- Most patients from age group 10–19 and 40–49
